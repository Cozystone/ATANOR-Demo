# ATANOR Launch Kit

This file is a public-facing messaging kit for introducing ATANOR.

## One-Line Description

ATANOR is a graph-native, local-first AI architecture for transparent memory, verifiable reasoning, and installable knowledge.

## Short Pitch

ATANOR asks a different question from mainstream AI products: what if memory, provenance, retrieval, repair, and shared knowledge were visible graph systems instead of hidden inside a remote black box?

It keeps private memory local, grows public knowledge as graph fragments, and lets graph cartridges attach read-only into working memory. The result is an alpha research platform for AI that can be inspected, audited, and improved without surrendering the user's private data.

## Vision Pitch

ATANOR is not only an app. It is a thesis about where personal AI should go next.

The future we want is not a single remote model that remembers everything for everyone. It is a memory architecture where private knowledge stays local, public knowledge moves as graph fragments, expertise can be installed as graph cartridges, and every important answer can be traced back to visible structure.

If AI becomes infrastructure, memory must become inspectable.

## Product Pillars

- **Local Brain:** private graph memory on the user's machine.
- **Cloud Brain:** public, proof-scale graph fragments and semantic growth.
- **Graph Hub:** installable graph cartridges, not prompt packs.
- **Working Memory Overlay:** temporary attachment with provenance boundaries.
- **Surface Brain:** answer quality, repair candidates, and review queues.
- **Q-Cortex / CORTEX-G2:** explicit salience, planning, activation, and predictive loops.
- **Proof Artifacts:** tests and proof snapshots for the claims we make.

# ATANOR SPLATRA Cosmos Cell

The SPLATRA Cosmos Cell is the wide universe behind bulletproof glass. Inside
it, the agent can run synthetic particle compression experiments, evaluate LOD
budgets, map emotion/audio controls, and propose orb patches. Outside it, every
action requires capability and approval.

Allowed paths are limited to the hologram orb/dashboard integration surface,
`packages/splatra_turbovec/**`, SPLATRA documentation, and ignored sandbox proof
outputs.

Limits:

- max cycles default `5`;
- max runtime default `300s`;
- max files changed default `8`;
- max diff lines default `600`;
- generated particle proof budget default `200k` in memory;
- no large assets committed;
- no direct commit or push.

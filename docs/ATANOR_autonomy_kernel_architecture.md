# ATANOR Autonomy Kernel Architecture

This document describes a proof-only autonomous self-model loop for ATANOR. It
does not claim real consciousness, AGI, or IIT proof.

## World Model

The World Model is a bounded snapshot of concepts, relations, evidence,
unresolved questions, contradictions, and confidence gaps. It is input data for
deficit calculation, not a production memory mutation path.

## Self Model

The Self Model is a bounded snapshot of local memory counts, user goals, recent
runs, resources, known limits, and active projects. It is used to reason about
what ATANOR can safely propose next.

## Deficit Signal

Deficit Signals are deterministic control signals derived from gaps,
contradictions, resource pressure, missing skills, and unresolved user goals.
They may be described metaphorically as itch-like control signals, but they are
not emotions.

## Self-Referential Loop

The proof loop is:

1. observe
2. compute deficit
3. plan
4. sandbox congress
5. generate proposal
6. await review
7. present morning brief

All proposals require review. The proof loop does not execute generated code,
mutate production, write Local Brain, or use real P2P networking.

## Midnight Congress

Midnight Congress is modeled only as a local sandbox simulator with synthetic
peer avatars. It creates reviewable arguments and proposal candidates without
network access, external LLMs, or private data transfer.

## Knowledge Cartridge Protocol

Knowledge Cartridge exchange is metadata-only in this proof. Raw payloads are
excluded by default. A future production path must pass through Tabularis for
private structured data and Atlas Router for safe public routing before any
temporary attach.

## Mutual Perception Event Stream

The event stream is local and in-memory. It can represent insights, warnings,
proposals, blocked states, deficit detection, congress summaries, and morning
briefs. There is no UI or API integration yet.

## Safety Gates

The proof enforces:

- no generated code execution
- no production mutation
- no Local Brain mutation
- no real P2P
- no network
- no raw private data export
- no consciousness claim

## Relation To Atlas Router And Tabularis

Atlas Router is the future routing layer for safe public paths. Tabularis is
the future privacy shield for structured private data. The Autonomy Kernel must
sit behind both before any broader autonomous planning can touch real data.

## Why Hot-Swap Is Proposal-Only

Self-generated code and hot-swap are not safe production behaviors at this
stage. They are reworded as proposal-only patch plans that require sandbox
tests, review, and explicit human approval.

## Future Integration Path

Integration should wait until the active 24h candidate-learning run completes,
candidate promotion gates are mature, privacy gates are explicit, and event
stream UX/API surfaces are designed as read-only review channels first.

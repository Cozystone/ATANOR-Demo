# 커넥톰 진화 생태계 — 검증 결과 (2026-07-07)

사장님의 재정의("추론기가 아니라 진화하는 메타 제어기")를 **격리 실험으로 측정**했다.
결론은 두 갈래로 정직하게 갈린다.

## 무엇을 만들었나
`packages/atanor_ecosystem/` — OBSERVE 전용 샌드박스. 답변 경로·스토어에 절대
배선 안 함. 스케줄링 정책 에이전트 집단이, **환각 0 가드레일을 적합도(천적)로**
진화한다: 좋은 스케줄링엔 보상, 거짓 팩트를 하나라도 주장하면 즉사(−1000).
커넥톰 초기화 vs 무작위 초기화 A/B 포함. 5 테스트, 결정론적.

## 검증된 것 (✅ 채택)
**메커니즘 = 진짜다.** 가드레일을 적합도로 쓰는 자연선택은 안정적으로 작동한다 —
정합적 스케줄러로 진화하고(적합도 ~1.0), 환각을 주입한 에이전트는 죽는다. 사장님이
지적한 "임의적 배선" 맹점을 선택으로 해소한 것이 옳았다. 이건 실현 가능한 안전
탐색 엔진이다. **설계 원리 발견**: 가혹한 천적은 gen-0을 몰살하므로, 생명은
**보수적으로 시작해 대담함을 획득**해야 한다(안전탐색 초기화). ATANOR의 비대칭과
같다.

## 반증된 것 (❌ 기각)
**커넥톰 위상 자체는 기능이 아니다 — 측정상 해롭다.** C. elegans 배선도로 초기화하면
4개 시드 전부에서 무작위 초기화보다 나빴다(Δ −0.33~−0.83). 경직된 생물 위상이
진화에 필요한 다양성을 줄여 나쁜 골짜기에 가둔다. 즉 **진화가 일을 하고, 커넥톰은
장식이며, 정책 구조로 쓰면 오히려 해가 된다.**

## 종합 (실현 방안)
- **지어라**: 진화형 메타 제어기(가드레일=적합도) — 자원 스케줄링을 스스로 학습하는
  격리 사이드카. **무작위/다양성 초기화**로. 고정 homeostasis 휴리스틱을 이기는지
  측정으로 증명한 뒤에만 내부 노브(소독 빈도·연상 가중)에 인간 게이트로 승격.
- **버려라**: 커넥톰을 정책 위상으로 쓰는 것. (원한다면 SPLATRA 시각화의 미적
  요소로만 — 답변 신뢰엔 절대 배선 금지.)
- **정직**: 실제 OpenWorm 302-뉴런 인제스트는 로더 배선 한 스텝(문서화)이지만,
  이 실험은 그 위상이 **기능적으로 무의미하거나 해롭다**는 걸 이미 측정했다.
  "해독된 생명을 이식해 배양"의 낭만은 이해하나, 데이터가 반증한다.

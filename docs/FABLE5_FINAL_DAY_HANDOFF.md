## 1. 무엇이 섰나 (트랙별)

### 로드맵 Phase 2~5 (오전)
- **2-5** 프로세스 샤드 워커 — 진짜 멀티코어 병합, 실측 x2.15(774→1,665 decomp/s)
- **3-2/3-3** 사용자 심층 모델 + 자의식-추론 융합
- **3-4** 내발적 커리큘럼 (가치 스택→학습 큐)
- **3-6/3-7/3-8** Qualia 씨앗 3종: 항상성+디지털 호르몬 / 다중감각 간섭 / 은유 위상공간
- **4-2/4-4/4-5** 지각 AGI: 외형→파티클 재현 / 시각-KG 앵커링+인스턴스 매칭 / 실시간 카메라
- **5** 티어 승격 추천 + 디바이스 연속성 + 4-5×3-6 지각→자의식 배선
- 3D 간섭 시각화 실버그 수리(hidden-tab rAF 정지)

### 난제 하드 트랙 (오후)
- **P1** 다의어 허브 분리 (sense induction + 3단 문맥 해소, answer_bridge 훅)
- **P2** ATANOR 브라우저 밑바탕: page_distiller → 다출처 합의 → 판정 게이트 →
  default-deny 승격 게이트 → 세션 오케스트레이션 + 네이티브 셸 계약
- **P3** atanor-shell M2 SPLATRA 컴포지터 정밀 계약
- **자가정제 플라이휠 완성**: 1단계 모순소독 + 1b 분류노이즈 + 2단계 압축 +
  3단계 가설민팅 + 건강 리포트(관측). 실측 무결성 1.0(2,526만 팩트)
- **터보 인제스트**: Arrow 파일직행 3.0M rows/s(x4.4), Gemini 명세는 원리만 채택

### AI 브라우저 (사장님 명확화 반영)
- 크롬형 웹 브라우저인데 모든 행위가 개인 브레인 그래프로. **2레인 분리**:
  내용=공유(합의 게이트) / 행위=개인(로컬 저널, PII 게이트). 세션 레이어 + 계약.

### 보안 (핵심 5종 완결)
- 위협 총정리 / peer_trust_guard(Sybil·암호신원·철회가능 격리) / PII 가드(탐지·격리·
  잊힐권리) / injection_guard(주입=데이터) / rate_limiter(DoS). HW ID 저주는 기각.

### 창작 융합
- holographic_lm 다음-토큰이 답변 경로에 배선 — "X 시 써줘"가 근거 창작(뼈+살+파동)

## 2. 라이브 활성화에 필요한 것 (운영자 게이트)

새 백엔드 모듈·로직은 전부 테스트로 검증됐으나, **:8502 백엔드는 이 세션 코드로
재기동되지 않았다**(운영자 전용 계약). 아래는 재기동 후 라이브된다:

- **새 API 라우트**(재기동 후 활성): `/api/browser/*`(ingest/promotable/promote-preview/
  forget/activity·session/*), `/api/base-brain/graph-health`, `/api/atanor/user-model`,
  `/api/perception/visual-ingest`, `/api/brain-link/peer/{challenge,appeal}`
- **자가정제 데몬 틱**(재기동 후 가동): ATANOR_SELF_REFINE_EVERY(240) — 모순/분류/압축/
  PII 소독 + 가설민팅. env로 조정 가능.
- **새 채팅 레인**(재기동 후): "나에 대해", "X는 뭐 같아?", "어떻게 생겼어", "X 시 써줘"

**운영자 액션**: named 재기동으로 :8502를 이 커밋(34c85daf)으로 올리면 위 전부 라이브.

# Digital Life Kernel v0

Digital Life Kernel v0 is a proof-only organism loop for ATANOR. It observes
bounded local state, emits typed life signals, proposes safe actions, runs those
proposals through a local sandbox, and records events.

It does not mutate production, write Local Brain, promote candidate stores, use
real P2P, execute generated code, or claim real consciousness or AGI. Every
action proposal requires review by default.

# ATANOR Surface Brain

Surface Brain is the expression layer for ATANOR. Semantic Cloud Graph decides what can be said; Surface Cloud Graph decides how to say it.

This package is local, deterministic, and proof-oriented. It does not use external LLMs or sLLMs, does not write to Local Brain, and does not claim GPT-level generation quality. Q-Cortex can optionally optimize construction, discourse, lemma, and repair candidate selection.

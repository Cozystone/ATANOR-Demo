# Graph Hub

Graph Hub is ATANOR's local, store-like system for Graph Cartridges.

A Graph Cartridge is a portable graph-native intelligence module. It can contain seed extensions,
semantic graph nodes and relations, surface graph constructions, reasoning patterns, repair rules,
evaluation prompts, license metadata, safety metadata, and entitlement metadata.

Current MVP boundaries:

- Local catalog only.
- Local/mock free, one-time, and subscription entitlement modes.
- No real payment processing.
- No remote marketplace backend.
- Default cartridge attachment is read-only.
- Cartridge data does not silently merge into Local Brain.
- Semantic Cloud exports come from the real proof store, not the old mirror snapshot.

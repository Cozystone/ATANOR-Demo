# Live Selfhood Life Signs Monitor

Proof-only observability for ATANOR's Live Selfhood Cycle.

The monitor reports functional life signs:

- heartbeat and tick age
- rhythm mode and wake reason
- latest observation, need, impulse, spark, proposed action, and brief
- pending user approvals
- safety blocks
- resting, stalled, stopped, idle, or alive status

It does not claim real consciousness, AGI, IIT proof, real P2P, cloud upload, Local Brain mutation, production mutation, candidate promotion, generated-code execution, always-on microphone capture, or raw voice storage.

Run the proof:

```powershell
python -m packages.live_selfhood_monitor.proof
```

Proof outputs are written under `data/audits/live_selfhood_monitor/` and should not be committed.

# ATANOR Base Brain Pack v0

Base Brain Pack v0 is a local, zero-user-data graph brain proof.

It combines:

- Seed Graph v2: primitive relations, reasoning moves, intents, and grounding states.
- Base Semantic Graph v0: a small curated general-purpose concept graph.
- Base Surface Graph v0: Korean and English construction candidates for Surface Brain.

It does not claim GPT-level quality, complete world knowledge, web-scale crawling,
or a trained neural decoder. It is a bounded proof that ATANOR can answer a small
general benchmark without user documents, external LLMs, external sLLMs, or web calls.

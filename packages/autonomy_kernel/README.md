# Autonomy Kernel

Autonomy Kernel is a proof-only package for ATANOR's future autonomous
self-model loop.

It models:

- World Model snapshots
- Self Model snapshots
- deterministic Deficit Signals
- proposal-only autonomous planning
- local sandbox Midnight Congress simulation
- metadata-only Knowledge Cartridge protocol
- Mutual Perception event stream
- morning brief generation

This package is not real consciousness, not AGI, not an IIT proof, and not
self-modifying production code. It does not create real P2P connections, does
not transfer real private data, does not mutate production stores, and does not
write Local Brain. Autonomous proposals always require approval.

Run tests:

```powershell
python -m pytest packages/autonomy_kernel/tests -q
```

Run proof:

```powershell
python -m packages.autonomy_kernel.proof
```

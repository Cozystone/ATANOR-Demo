# SPLATRA Imagination Field

Procedural particle imagination for ATANOR's hologram surface.

This package creates deterministic visual archetypes such as orb, tower, tree, creature, circuit, city block, constellation, machine core, and abstract memory cloud. These are visualization frames only. They are not verified facts, not Local Brain memory, and not Cloud Brain production knowledge.

Safety defaults:

- no external LLM or sLLM
- no image generation model
- no Local Brain write
- no production store mutation
- no candidate promotion
- no generated scene committed
- proof-only by default
